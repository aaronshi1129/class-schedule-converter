How to Import Your Schedule

Google Calendar

  1. Export your schedule as CSV or .ics file
  2. Go to Google Calendar
  3. Click on the gear icon ⚙️, then "Settings"
  4. Click "Import & Export" on the left
  5. Click "Select file from your computer" and choose your CSV file
  6. Select the calendar you want to import to
  7. Click "Import"

Apple Calendar

  1. Export your schedule as .ics file
  2. Open the Calendar app on your Apple device
  3. Go to File > Import
  4. Select the .ics file you exported
  5. Click "Import"

Outlook

  1. Export your schedule as .ics file
  2. Open Outlook
  3. Go to File > Open & Export > Import/Export
  4. Select "Import an iCalendar (.ics) or vCalendar file (.vcs)"
  5. Browse to your .ics file and open it
  6. Choose whether to add to your calendar or create a new calendar
